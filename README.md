## How to use

### Project set up

` git clone git@github.com:miochung7/emojify.git`

### Database set up

- Connect to `psql`
- Create the `registration` database:
```
CREATE DATABASE registration;
```

### To run the app:

```
mvn spring-boot:run
```

Navigate to `localhost:8080/signup`
